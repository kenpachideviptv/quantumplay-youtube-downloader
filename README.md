# 🚀 QuantumPlay - YouTube Downloader

**QuantumPlay** é um aplicativo desenvolvido em **Java 21** para baixar e converter vídeos e áudios do YouTube de forma rápida, prática e intuitiva. O programa integra automaticamente **yt-dlp** e **FFmpeg**, dispensando configurações manuais.

## ✨ Recursos

* 🎬 Download de vídeos em alta qualidade
* 🎵 Conversão para MP3 e WAV
* 📹 Download em MP4
* ⚡ Download rápido e eficiente
* 🛠️ yt-dlp integrado
* 🎞️ FFmpeg integrado
* 📂 Organização automática dos downloads
* 🖥️ Interface moderna e fácil de usar

---

## 📋 Requisitos

* ☕ Java 21 (ou superior)
* 💻 Windows 10 / Windows 11
* 🌐 Conexão com a Internet

### 📥 Download do Java 21

Baixe o **Java JDK 21** no site oficial da Oracle:

👉 https://www.oracle.com/java/technologies/javase/jdk21-archive-downloads.html

## 🚀 Como executar

1. Instale o Java 21.
2. Execute o arquivo **QuantumPlay.jar**.
3. Cole o link do vídeo.
4. Escolha o formato desejado.
5. Clique em **Baixar**.
